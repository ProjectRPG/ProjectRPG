# Getting Started for Developers

## Overview
Project RPG is a framework which lets users pick and choose the RPG elements they want in their world by selecting Project RPG addon mods which have the features they want.  The elements users can customize are as follows

|System|Purpose|
|:---:|:---|
|Progression| How players advance |
|Progression Addons| Modifiers for Advancement |
|Gates| Prevents or limits gameplay for players |
|Abilities| special abilities players have or use|
|Features| Hooks into Project RPG for non-RPG content|
|Parties|Player grouping mechanics|
|Events|Triggers for RPG behavior|

It is strongly recommended that a mod implement only one of these elements.  The framework is intended for players to swap out addons as needed to get the specific experience they desire.  An addon which implements more than one element may contain a feature the user wants and one they don't, which undermines the user choice that this framework aims to provide.  If you do consolidated elements into a single mod, it is strongly advised that you provide configuration options allowing individual parts to be disabled.
