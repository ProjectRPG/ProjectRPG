# Examples

Code examples for implementing Project RPG addons.

## Custom Ability
```java
public class FireballAbility extends Ability {
    public FireballAbility() {
        super(
            (p, s, c) -> true, // conditions
            new CompoundTag(), // defaults
            (p, s, c) -> { /* start logic */ return new CompoundTag(); },
            (p, s, c, t) -> { /* tick logic */ },
            (p, s, c) -> { /* stop logic */ },
            Component.literal("Shoots a fireball"),
            (p, s, c) -> List.of(Component.literal("Damage: 10")),
            RegistrationSide.BOTH
        );
    }
}
```

## Registering in Mod
```java
@Mod("exampleaddon")
public class ExampleAddon {
    public ExampleAddon(IEventBus bus) {
        CommonSetup.ABILITIES.register(bus);
        bus.addListener(this::registerAbilities);
    }

    private void registerAbilities(RegisterEvent event) {
        event.register(Identifier.of("example", "fireball"), new FireballAbility());
    }
}
```
