# Project RPG Wiki

Welcome to the Project RPG documentation! Project RPG is a hub mod for Minecraft that enables RPG (Role-Playing Game) mechanics through a flexible framework. It acts as a foundation for addons to introduce abilities, progression systems, gating, features, and more, allowing players to experience customized RPG elements in their Minecraft worlds.

This wiki is divided into two main sections:

- **[User Guide](users/)**: For players and server administrators who want to understand and use Project RPG and its addons.
- **[Developer Guide](developers/)**: For mod developers interested in creating addons that extend Project RPG's capabilities.

## Quick Start
1. Install NeoForge for Minecraft 1.21.1.
2. Download Project RPG from [CurseForge/releases].
3. Add compatible addons to your mods folder.
4. Launch Minecraft and configure as needed.

For more details, see the [Installation Guide](users/Installation.mdx).

## Features Overview
Project RPG provides:
- **Abilities**: Special actions players can activate under conditions.
- **Progression**: Systems for tracking player advancement.
- **Gating**: Restrictions on actions based on progress.
- **Features**: Custom event callbacks.
- **Events**: Triggers for RPG mechanics.
- **Party System**: Group-based interactions.

Addons can implement these to create unique RPG experiences.
