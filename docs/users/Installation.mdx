# Project RPG

Project RPG is a framework mod for Minecraft that introduces RPG elements through a modular addon system.

## Base Functionality

Project RPG provides base RPG mechanics out of the box, including default implementations of abilities, progression, gating, features, and events. These serve as placeholders and can be overridden by installing compatible addons.

See the [Default Implementations](defaults/) for details:
- [Abilities](defaults/Abilities.mdx)
- [Events](defaults/Events.mdx)
- [Bonuses](defaults/Bonuses.mdx)
- [Party System](defaults/Party.mdx)
- [Progression](defaults/Progression.mdx)

## Addons

To extend or customize the RPG experience, install addons that integrate with Project RPG. Addons can provide custom abilities, progression systems, and more.

Find addons on platforms like CurseForge or Modrinth.
