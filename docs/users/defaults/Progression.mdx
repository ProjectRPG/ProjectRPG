# Progression System

The default progression system uses Minecraft's experience levels to track player advancement.

- **Tracking Progress**: Player's experience level represents their progress in the "exp" container.
- **Awarding XP**: Awards experience points from configured events and subjects, applying bonuses from addons.
- **Offline Progress**: Caches progress for offline players and applies on login.
- **Commands**: Provides /xp set and /xp add commands for administrators.
- **Side Panel**: Displays progression info in a UI panel.
