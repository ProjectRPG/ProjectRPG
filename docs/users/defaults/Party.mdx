# Party System

The default party system uses Minecraft's scoreboard teams to manage player groups.

- **Creating Parties**: Creates a new team for the party.
- **Inviting Players**: Sends invites by adding players to pending invites list.
- **Accepting/Declining Invites**: Manages invite responses and team membership.
- **Joining/Leaving Parties**: Handles team membership changes, auto-closes empty parties.
- **Party Info**: Retrieves party names, members, and display names from teams.
