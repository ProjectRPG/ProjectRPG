# Project RPG

Project RPG is a hub mod for Minecraft that enables RPG mechanics through addons. It provides a framework for abilities, progression, gating, features, events, and parties.

See [docs/users/Installation.mdx](docs/users/Installation.mdx) for setup instructions.
